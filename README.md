# EasyNotes
EasyNotes helps you organize your notes efficiently.<br>
<br>
It provides functionality to:<br>
-> Add a new note by clicking AddNote.<br>
-> Edit the existing note.<br>
-> Deleting a note.<br>
<br>
Notes are being stored in the Database.
